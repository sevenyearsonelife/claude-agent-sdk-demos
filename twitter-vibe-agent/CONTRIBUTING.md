# 贡献指南

欢迎贡献！为了保持项目质量与一致性，请遵循以下约定。

## 贡献范围
- 文档改进（README、教程、示例）
- 提示词优化（prompts 与 skills）
- 代理行为改进与可观测性增强
- Bug 修复与小范围功能增强

## 开发流程
1. 新建分支并进行小步提交
2. 保持变更范围最小、目的明确
3. 更新相关文档，说明变更动机与影响

## 提交前检查
- 运行语法检查：
  ```bash
  python -m compileall twitter-vibe-agent
  ```
- 确认示例路径与输出结构未被破坏

## 代码风格
- 优先保持与现有代码一致
- 不做无关重构
- 注释简洁且必要

## 讨论与问题反馈
- 功能建议与问题报告请通过 Issue 提交
- 重大变更建议先开 Issue 讨论再提交 PR
